HTML Advanced Text Tags

Definition List

Element Name Description
<dl> Description list The wrapper for an HTML description list
<dt> Description term The container for the term being defined
<dd> Description definition The container for the definition of the term being defined

Quotations

Element Name Description
<blockquote> Blockquote A container for an extended quotation
<q> In-line quote A container for short in-line quotations without paragraph breaks
<cite> Cite Contains the title of the resource being quoted

Abbreviations

Element Name Description
<abbr> Abbreviation Container to hold abbreviations
Attribute Name Description
title Title Provides full expansion of the term

Contact Details

Element Name Description
<address> Contact details Container wraps around contact details

Superscript and Subscript

Element Name Description
<sup> Superscript Wraps text that will be raised to Superscript
<sub> Subscript Wraps text that will be lowered to Subscript

Representing Computer Code

Element Name Description
<code> Code For marking up computer code
<pre> Preformatted Wraps preformatted text
<var> Variable For marking up variable names
<kbd> Keyboard For marking up keyboard input
<samp> Sample For marking up sample of program output

Times and Dates

Element Name Description
<time> Time Mark up times in machine readable format
Attribute Name Description
datetime Date Time Contains machine readable time
